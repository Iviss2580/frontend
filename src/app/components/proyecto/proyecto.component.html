Creacion de portfolio web

Es el trabajo final de Argentina Programa para desarrolladores web, donde se crea un front end (usando Visual Studio Code) que luego se relaciona con su back end (usando Apache NeatBeans) y se conecta con una base de datos (MySQL). Tiene un login, seguridad cibernetica a traves de un token. Cada avance es guardado en el repositorio de Git Hub

Abundance

Es un proyecto del universo criptomonedas, especificamente del trading automatico llevado adelante por inteligencia artificial, en este proyecto se crea un propio indicador llamado "Abundance", que contiene distintas variables e indica momentos de compra y de venta.
